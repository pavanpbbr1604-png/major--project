Project Overview

Crowd Density Estimation using Multi-Perspective Image Analysis is a modern computer vision pipeline built to estimate crowd densities and counts under complex, highly crowded scenarios.

Key Objectives

  • Person Detection: Employs state-of-the-art YOLOv8x deep learning architecture for robust object classification.
  • Crowd Density Estimation: Uses geometric binary occupancy masking algorithms to merge overlapping areas, preventing false estimations.
  • Multi-Perspective Fusion: Harmonizes multiple viewing angles to build consolidated consensus estimates without duplicate counts.
  • Crowd Classification: Integrates counts and mask areas to generate crowd classification ratings.

Technologies Employed

Python Flask PyTorch YOLOv8x (Ultralytics) OpenCV (cv2) SQLite HTML5 & Vanilla CSS
